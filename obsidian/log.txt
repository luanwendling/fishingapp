  Feature: feature/ai-detection-measurement-v1

  Objetivo:
  Criar o motor de IA offline responsável por medir o comprimento de um peixe a partir de um vídeo (.mp4).

  ⚠️ Ainda não existe nenhuma estrutura criada.
  Você deve criar o projeto do zero com arquitetura modular e escalável.

  -------------------------------------------------
  🎯 OBJETIVO DO MVP

  Criar um pipeline Python que:

  1. Receba um vídeo (.mp4)
  2. Extraia frames
  3. Detecte dois objetos usando YOLOv8:
     - fish
     - ruler
  4. Calcule proporção pixel → cm
  5. Meça o comprimento horizontal do peixe
  6. Gere JSON final:

  {
    "length_cm": float,
    "confidence_score": float,
    "status": "review"
  }

  O status deve ser sempre "review".

  -------------------------------------------------
  📁 ESTRUTURA DO PROJETO

  Criar a seguinte estrutura:

  fishing-ai-engine/

  │
  ├── tests/

  ├── requirements.txt
  ├── README.md
  └── .gitignore

  -------------------------------------------------
  🧠 RESPONSABILIDADES DOS ARQUIVOS

  video_processor.py:
  - Abrir vídeo com OpenCV
  - Extrair frames (ex: 1 frame a cada X ms)
  - Retornar lista de frames

  detector.py:
  - Carregar modelo YOLOv8
  - Detectar fish e ruler
  - Retornar lista padronizada:
    [
      {
        "class": "fish",
        "confidence": 0.93,
        "bbox": [x1, y1, x2, y2]
      }
    ]

  measurement.py:
  - Calcular largura da régua em pixels
  - Converter para cm
  - Calcular comprimento do peixe

  confidence.py:
  - Combinar:
     - confiança YOLO
     - variação entre frames
  - Retornar confidence_score final

  main.py:
  - Orquestrar pipeline completo
  - Permitir execução via CLI:
    python -m ai.main --video path.mp4

  -------------------------------------------------
  📏 REGRAS DE MEDIÇÃO

  - Régua possui tamanho fixo (definir como constante no config.py)
  - Medição baseada na largura horizontal do bounding box
  - Usar média dos melhores frames

  def _calculate_pixels_per_cm_from_ticks(
      self,
      frame: np.ndarray,
      ruler_bbox: tuple[float, float, float, float],
  ) -> float | None:
      import numpy as np
      import cv2

      h, w = frame.shape[:2]
      x1, y1, x2, y2 = [int(v) for v in ruler_bbox]

      x1 = max(0, min(x1, w - 1))
      y1 = max(0, min(y1, h - 1))
      x2 = max(1, min(x2, w))
      y2 = max(1, min(y2, h))

      if x2 <= x1 or y2 <= y1:
          return None

      ruler = frame[y1:y2, x1:x2]
      if ruler.size == 0:
          return None

      # Converter para escala de cinza
      gray = cv2.cvtColor(ruler, cv2.COLOR_BGR2GRAY)

      # Suavização leve
      gray = cv2.GaussianBlur(gray, (5, 5), 0)

      # Realçar marcas escuras
      normalized = cv2.normalize(gray, None, 0, 255, cv2.NORM_MINMAX)

      # Projeção vertical (soma por coluna)
      projection = np.mean(normalized, axis=0)

      # Inverter sinal (marcas são mais escuras)
      projection = np.max(projection) - projection

      # Remover tendência (centralizar)
      projection = projection - np.mean(projection)

      # Autocorrelação
      autocorr = np.correlate(projection, projection, mode="full")
      autocorr = autocorr[len(autocorr)//2:]

      # Ignorar primeiros pixels (lag 0)
      min_lag = 5
      autocorr[:min_lag] = 0

      # Encontrar pico dominante
      peak_index = np.argmax(autocorr)

      if peak_index <= 0:
          return None

      pixels_per_cm = float(peak_index)

      # Sanity check
      if pixels_per_cm < 5 or pixels_per_cm > (ruler.shape[1] / 2):
          return None

      return pixels_per_cm
